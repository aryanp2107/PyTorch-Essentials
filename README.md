# PyTorch Learning
 
A collection of notebooks and scripts documenting my PyTorch learning journey — covering core concepts, neural network architectures, and hands-on experiments.
 
## Topics Covered
 
- Tensors, autograd, and computational graphs
- Building neural networks with `torch.nn`
- Training loops, loss functions, and optimizers
- CNNs for image classification
- Data loading and transforms with `torchvision`
- GPU acceleration with CUDA
- Model saving and loading
 
## Tech Stack
 
- Python 3.x
- PyTorch
- Torchvision
- Matplotlib / NumPy
- Google Colab / Jupyter Notebooks
 
## How to Use
 
1. Clone the repo:
   ```bash
   git clone https://github.com/aryanp2107/pytorch-learning.git
   ```
2. Install dependencies:
   ```bash
   pip install torch torchvision matplotlib numpy
   ```
3. Open any notebook and run the cells sequentially.
